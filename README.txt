
Este código tem o objetivo de desenvolver uma comunicação entre a Plataforma Arduino e o software ScadaBR, utilizando o protocolo ModBus.

As versões dos Softwares utilizadas:

	Arduino IDE 1.6.9
	ScadaBR 1.0
	JDK 6

O código executou sem problemas na versão UNO e MEGA do Arduino.

Nessa pasta contém os arquivos "sketch_jun22a.ino" (principal), "modbus.h" (Biblioteca do ModBus para Arduino), o "esquema_eletrico.png", "background.png", e o backup com as configurações do ScadaBR "backup_config_scadaBr.zip".
